# Installation
Clone the repository into your kak config location
```
$ mkdir ~/.config && git clone https://github.com/jenterkin/kakconfig.git ~/.config/kak
```

Install plug.kak
```
$ mkdir ~/.config/kak/plugins && git clone https://github.com/andreyorst/plug.kak.git ~/.config/kak/plugins/plug.kak
```

Install the plugins by opening `kak` and running `<esc>:plug-install`.
